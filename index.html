JavaScript turn based game.

Using Vanilla JavaScript, created a turn-based online game in which two players move around the board, collecting weapons and avoiding obstacles. The goal of the game is to destroy the other player , on its turn a player could either move, attack or defend. Pure JavaScript was used for creating the board and the game itself, the styling has been done using custom CSS.

Responsive Webpage Design for Movie Festival

Using HTML, CSS and jQuery designed a responsive website for “Watch it Outside Events Co” to publicize their annual Film Festival. The website consists of an information about the festival, schedule of the movies projected, regularly updated news about the event as well as links to their Facebook and Instagram accounts. Furthermore, to give the website a more professional look and performance added a subscribe and register form as well as information about means of transport to and from the festival. For the responsiveness of the designed Bootstrap was used and all of the styling was made using CSS.
